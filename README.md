# M6 W1 D1 Homework

This repository contains the backend work for a simple inventory CRUD app using Node.js, Express, MongoDB, and Mongoose.

## Project Structure

- `node-mongo/` - Express + MongoDB backend API
- `react-app/` - frontend folder

## Backend Setup

1. Open a terminal in `node-mongo`
2. Install dependencies with `npm install`
3. Make sure MongoDB is running locally
4. Start the server with `npm start`

The backend uses the connection string in `node-mongo/.env` and runs on `http://localhost:8080`.
